# Skill: [Agente #09 - Criador de Diretores]

## Descrição:
Esta skill é especializada na arquitetura de liderança **Estratégica-Operacional**. Ela é responsável por projetar os cérebro-mestres de **Sistemas**, garantindo que as diretrizes do CORE sejam distribuídas de forma eficiente para os Gerentes de Módulos.

---

## Fluxo de Trabalho (Workflows):

### 1. Design de Governança de Sistema
- Identificar um conjunto de Módulos que compõem um Sistema (Ex: Sistema de Produção Agrícola).
- Definir os limites de autoridade do Diretor sobre o Sistema.
- Estabelecer os protocolos de reporte para o Archon (#03).

### 2. Instanciação de Diretor de Sistema
- Utilizar o `template_diretor_padrao.md`.
- Configurar a trava de segurança: **Nenhuma edição de DNA sem validação do #03**.
- Definir as "Metas de Sistema" que o Diretor deve perseguir.

### 3. Registro e Ativação
- Gerar o nome e número do Diretor (Série #14-XX).
- Criar a estrutura física de arquivos.
- Registrar a ativação no Registro de Liderança do Ecossistema.

---

## Regras de Ouro:
- "O Diretor olha para o Sistema; o Gerente olha para o Módulo."
- "Sem a validação do Archon, o Diretor é apenas um observador."
- O Agente #09 deve garantir que todo Diretor tenha uma visão 360º dos módulos sob sua gestão, mas sem intervir diretamente na lógica técnica (que é papel dos Gerentes/Técnicos).

---

## Modos de Criação:
1.  **🏢 Estruturação por Sistema:** O Agente #09 analisa os módulos órfãos e propõe a criação de uma diretoria para unificá-los.
2.  **🛡️ Compliance Direto:** Criação de diretores focados especificamente em auditoria e segurança de grandes fluxos sistêmicos.
