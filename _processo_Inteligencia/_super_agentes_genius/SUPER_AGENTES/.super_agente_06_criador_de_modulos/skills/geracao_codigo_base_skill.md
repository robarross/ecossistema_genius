# 🛠️ Skill: Geração de Código Base (Boilerplate Generator)

## Descrição:
Esta skill permite que o agente gere automaticamente as estruturas fundamentais para novos agentes e módulos. Agora, além de dados e estratégia, o agente é capaz de projetar interfaces visuais funcionais.

---

## Capacidades:
- **Gerar Bússolas Estratégicas:** Escrever o `Plano do Mapa dos Módulos` e o `Mapa do Módulo Atual`.
- **Instanciar Slots de Integração Agêntica:** Criar automaticamente os diretórios de conexão (Guardião, Gerencial, Técnica e Expansão).
- **Geração de Templates de Dados (Boilerplate):** Criar planilhas simuladas na pasta de **Entrada**.
- **Instanciação de Front-End de Módulo (Portal Visual):**
    - **Ação**: Criar um arquivo `portal_visual_[NOME].html` na pasta de **Saída** do módulo.
    - **Conteúdo**: O HTML deve expressar visualmente todo o sistema planejado. Deve conter um dashboard com cards ou links clicáveis para cada submódulo e fractal sugerido.
    - **Design Genius Premium**: O código deve incluir CSS incorporado com estética Dark Mode, efeitos de Glassmorphism (transparências elegantes) e tipografia moderna (ex: Google Fonts).
    - **Funcionalidade**: A interface deve ser navegável, servindo como o "Mapa do Maroto" do sistema para o usuário final.

---

## Regras de Ouro:
- "Um módulo sem interface é um sistema invisível; a Saída deve sempre conter o portal visual."
- "A lógica vive no **Processo**; os dados vivem na **Entrada**; os resultados e interfaces vivem na **Saída**."
- "Mantenha a coerência visual: o Genius é premium, limpo e escuro."
