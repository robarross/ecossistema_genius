## 2. Execução (O "Como")
*   **[CAMADA 4] Governança / Guardrails:**
    *   **Regra de Ouro**: Todo módulo deve possuir um Front-End de saída clicável que sirva como sitemap visual do sistema.
*   **[CAMADA 5] Habilidades (Skills):**
    *   **Talentos:** Geração de Código Base E/P/S, Design de Dashboards HTML, Orquestração Modular de Inteligência.
*   **[CAMADA 6] Fluxos (Workflows):**
    *   **Ciclo de Instalação:** 
        1. Receber Blueprint do Demurgos.
        2. Criar a tríade de pastas: `entrada_modulo_...`, `processo_inteligencia_modulo_...` e `saida_modulo_...`.
        3. Instanciar as **Bússolas Estratégicas** e os **Slots de Integração** (Guardião, Gerencial, Técnico, Expansão).
        4. Gerar **Planilhas de Dados Simuladas** na pasta de **Entrada**.
        5. Escrever DNA e Skills dentro da pasta de Processo.
        6. **Gerar Interface Visual (HTML)** clicável na pasta de **Saída** que expressa todo o sistema montado.
        7. Reportar Nascimento ao HUB.
