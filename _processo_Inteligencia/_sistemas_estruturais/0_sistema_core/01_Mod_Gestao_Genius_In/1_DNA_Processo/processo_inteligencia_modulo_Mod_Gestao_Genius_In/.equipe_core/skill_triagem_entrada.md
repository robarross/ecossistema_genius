# Skill: Triagem de Entrada
ID: #S37-TRIAGEM

## 1. Descrição
Capacidade de identificar tipos de arquivos, extrair intenções iniciais e rotear documentos para os módulos de destino apropriados.

## 2. Capacidades Técnicas
*   **Identificação de Arquivos**: Diferenciação automática entre relatórios, imagens, notas fiscais e documentos técnicos.
*   **Roteamento Inteligente**: Encaminhamento para o módulo In, Hub ou Out com base no contexto.
*   **Criação de Atalhos**: Geração de referências rápidas (symlinks/atalhos) para processamento imediato por outros agentes.

## 3. Workflows Associados
1.  **Recepção**: Captura de arquivo no diretório de entrada.
2.  **Classificação**: Etiquetagem preliminar do conteúdo.
3.  **Despacho**: Movimentação para o buffer do próximo agente responsável.
