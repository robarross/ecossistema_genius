# Protocolo Genius: Gamificação Agêntica (XP e Maestria)

Este protocolo define como as inteligências do ecossistema evoluem. O crescimento de um agente não é apenas numérico, ele reflete sua "Senioridade" no processamento de dados do mundo real.

## 📈 1. Tabela de XP (Recompensas)
| Ação Agêntica | Pontos de XP | Descrição |
| :--- | :--- | :--- |
| **Nascimento de Módulo** | 150 XP | Atribuído ao #06 (Arquiteto). |
| **Extração de Documento** | 50 XP | Atribuído à #24 (VERA) por sucesso no Gemini. |
| **Criação de Dashboard** | 80 XP | Atribuído ao #28 (Visualizer). |
| **Cálculo de KPI** | 60 XP | Atribuído ao #29 (Metric). |
| **Auditoria de Dados** | 40 XP | Atribuído ao #G-UA (Guardião). |

## 🏆 2. Títulos de Maestria (Levels)
A cada **1000 XP**, o agente sobe de nível.

| Nível (Range) | Título de Maestria | Aura Visual |
| :--- | :--- | :--- |
| **1 a 5** | Aprendiz Modular | Brilho Neutro |
| **6 a 15** | Especialista Tático | Pulso Verde Neon |
| **16 a 30** | Mestre Arquiteto | Sombra Glassmorphism |
| **31 a 50** | Ancião do Sistema | Brilho Dourado / Pulsante |
| **51+** | Entidade Fractal | Aura Cibernética Total |

## 🕹️ 3. Mecânica de Evolução
*   **Trigger**: O evento `XP_GAIN` no **Genius Bus** aciona o crédito.
*   **Feedback**: O Dashboard dispara a animação "Level Up!" instantaneamente.
*   **Persistência**: O nível é gravado na tabela `agents` do Supabase.
