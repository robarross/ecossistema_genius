# 🎨 Diretriz Estética: Padrão Cyber-Ouro
Esta diretriz define a identidade visual futurista e de alta tecnologia do Ecossistema Genius.

## Princípios de Design:
1. **Cyber Dark Mode**: Fundo preto absoluto (#050505).
2. **Neon Glow (Azul & Violeta)**: Uso de brilhos suaves em Azul Neon para dados primários e Violeta Elétrico para alertas e destaques.
3. **Hardware Prata**: Ícones, divisores e tipografia técnica em Prata Cromado e tons de cinza metálico.
4. **Holographic Glassmorphism**: Cartões translúcidos com efeitos de blur e bordas luminosas.
5. **Estética Aerospacial**: Interfaces limpas, de alta precisão e densidade de informação inteligente.

## Aplicação:
- Todos os Dashboards gerados pelo Agente #06 devem seguir este padrão Cyber-Ouro.
