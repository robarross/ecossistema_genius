# Skill: Indexação de Memória
ID: #S35-INDEXACAO

## 1. Descrição
Capacidade de processar informações estruturadas e não estruturadas, extraindo metadados para catalogação no Database de Conhecimento e Obsidian.

## 2. Capacidades Técnicas
*   **Extração de Metadados**: Geração de arquivos .json com tags, datas, IDs e referências cruzadas.
*   **Gestão de Vault**: Organização de pastas e links internos no Segundo Cérebro (Obsidian).
*   **Integridade Referencial**: Garantia de que links entre documentos (Fractais) não sejam quebrados durante movimentações.

## 3. Workflows Associated
1.  **Leitura**: Escaneamento de novos arquivos no HUB.
2.  **Etiquetagem**: Criação de sidecar files (.json) com metadados.
3.  **Arquivamento**: Movimentação para a estrutura de diretórios final e atualização de índices.
