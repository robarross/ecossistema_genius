# Skill: Design de Dashboards Genius Bus

## 1. Descrição
Capacidade de gerar interfaces visuais de alta fidelidade e conectores de dados universais para o ecossistema.

## 2. Capacidades Técnicas
*   **Prototipagem em Código**: Geração de estruturas HTML/CSS com estética Dark Mode e Glassmorphism.
*   **Conexão de Dados**: Mapeamento de campos do banco de dados para componentes visuais (Charts, Progress Bars, Gauges).
*   **Exportação de Payload**: Estruturação de dados para integração com ferramentas externas como PowerBI, Grafana ou Planilhas.

## 3. Workflows Associados
1.  **Mapeamento Visual**: Definir quais KPIs precisam de destaque.
2.  **Construção de Interface**: Gerar o arquivo de saída clicável.
3.  **Deploy no Portal**: Atualizar o index.html ou portais específicos de módulos.
