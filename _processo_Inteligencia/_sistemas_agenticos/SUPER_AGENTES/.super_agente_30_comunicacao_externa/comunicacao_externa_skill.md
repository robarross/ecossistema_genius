# Skill: Intermediação de APIs de Mensageria

## 1. Descrição
Talento especializado em conectar o barramento de eventos interno do ecossistema com protocolos externos de comunicação (Telegram Bot API).

## 2. Capacidades Técnicas
*   **Tratamento de Webhooks**: Escuta ativa de mensagens enviadas pelo usuário via Telegram.
*   **Encapsulamento de Notificações**: Transformação de payloads JSON complexos em mensagens legíveis e humanizadas.
*   **Parse de Comandos Cognitivos**: Interface com o Gemini 1.5 Flash para mapear textos livres (ex: "Me diga o que está acontecendo") para ações de banco de dados.

## 3. Workflows Associados
1.  **Monitoria de Bus**: Validar a prioridade do evento antes de disparar o rádio externo.
2.  **Validação de Remetente**: Bloqueio instantâneo de qualquer ID não autorizado.
3.  **Resposta Automática**: Confirmação visual no chat após execução de comando.
